# Phonebook

This repository now contains both parts of the application.

- Backend: repository root
- Frontend: [frontend](frontend)

Run the backend from the repository root with `npm install` and `npm run dev`.
Run the frontend from [frontend](frontend) with `npm install` and `npm run dev`.

The frontend builds into the root `dist/` directory so the backend can serve it in production.

Pull Requests need to be approved by one other person.
